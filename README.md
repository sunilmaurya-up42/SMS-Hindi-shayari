# SMS Hindi Shayari

A modern and SEO-friendly Hindi Shayari platform built with Node.js, Express.js, MongoDB, EJS and Bootstrap.

---

## Features

- Beautiful Hindi Shayari Collection
- Category Wise Shayari
- Search Shayari
- Copy Shayari
- Share Shayari
- Download Shayari Images
- AI Generated Background Images
- Admin Dashboard
- User Authentication
- Responsive Design
- SEO Optimized
- XML Sitemap
- robots.txt
- PWA Support
- Google Analytics Ready
- AdSense Ready

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap 5
- JavaScript
- HTML5
- CSS3

---

## Installation

Clone Repository

```bash
git clone https://github.com/sunilmaurya-up42/SMS-Hindi-shayari.git
```

Open Project

```bash
cd SMS-Hindi-shayari
```

Install Dependencies

```bash
npm install
```
