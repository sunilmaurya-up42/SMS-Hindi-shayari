# SMS Hindi Shayari

A modern and SEO-friendly Hindi Shayari platform built with Node.js, Express.js, MongoDB, EJS and Bootstrap.

---

## Features

- Beautiful Hindi Shayari Collection
- Category Wise Shayari
- Search Shayari
- Copy Shayari
- Share Shayari
- Download Shayari Images
- AI Generated Background Images
- Admin Dashboard
- User Authentication
- Responsive Design
- SEO Optimized
- XML Sitemap
- robots.txt
- PWA Support
- Google Analytics Ready
- AdSense Ready

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap 5
- JavaScript
- HTML5
- CSS3

## Installation

Clone Repository

```bash
git clone https://github.com/sunilmaurya-up42/SMS-Hindi-shayari.git
```

Open Project

```bash
cd SMS-Hindi-shayari
```

Install Dependencies

```bash
npm install
```
---

## Environment Variables

Create a `.env` file in the project root.

```env
PORT=5000
NODE_ENV=development

MONGODB_URI=your_mongodb_connection_string

SESSION_SECRET=your_session_secret

JWT_SECRET=your_jwt_secret

SITE_URL=https://your-domain.com

GITHUB_OWNER=your_github_username
GITHUB_REPO=your_repository
GITHUB_BRANCH=main
GITHUB_TOKEN=your_github_token

ADMIN_NAME=Administrator
ADMIN_USERNAME=admin
ADMIN_EMAIL=admin@example.com
ADMIN_PASSWORD=StrongPassword

GOOGLE_ANALYTICS_ID=G-XXXXXXXXXX

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASS=
SMTP_FROM=
```

## Production Deployment

### Render

1. Create a new Web Service.
2. Connect your GitHub repository.
3. Add all required environment variables.
4. Set the build command:

```bash
npm install
```

5. Set the start command:

```bash
npm start
```
